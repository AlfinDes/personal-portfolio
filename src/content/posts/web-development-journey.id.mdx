Belajar web development mengajarkan saya bahwa membangun sebuah website bukan hanya soal tampilan, tetapi juga tentang bagaimana logika dan alur data bekerja di balik layar. Perjalanan saya dimulai dari HTML dan CSS, lalu berkembang ke JavaScript dan fundamental backend menggunakan Express.js.

Dalam artikel ini, saya membagikan pengalaman dan pemahaman yang saya dapatkan selama proses belajar tersebut.

## Fondasi Website dengan HTML dan CSS

HTML dan CSS menjadi dasar utama dalam setiap project yang saya kerjakan. Saya mempelajari bagaimana menyusun struktur halaman yang rapi menggunakan semantic HTML serta mengatur tampilan agar konsisten dan responsif menggunakan CSS.

Beberapa hal yang saya fokuskan:

- Struktur layout yang jelas dan mudah dipahami
- Responsive design untuk berbagai ukuran layar
- Penulisan CSS yang terorganisir dan mudah di-maintain

Pendekatan ini membantu saya membangun tampilan website yang bersih dan profesional.

## Membuat Website Lebih Hidup dengan JavaScript

Setelah memahami tampilan, saya mulai menambahkan interaktivitas menggunakan JavaScript. Di tahap ini, saya belajar bagaimana website dapat merespons aksi pengguna secara real-time.

Contoh implementasi yang saya pelajari:

- Manipulasi DOM untuk mengubah konten secara dinamis
- Event handling seperti klik dan submit
- Validasi form sederhana
- Logika dasar untuk meningkatkan pengalaman pengguna

JavaScript membuat website yang saya bangun terasa lebih hidup dan fungsional.

## Mengenal Backend melalui Fundamental Express.js

Untuk memahami bagaimana frontend terhubung dengan server, saya mulai mempelajari fundamental Express.js. Di tahap ini, saya belajar:

- Membuat server sederhana dengan Node.js
- Routing dasar menggunakan Express
- Cara menangani request dan response
- Konsep REST API sederhana

Walaupun masih di tahap fundamental, Express.js membantu saya memahami gambaran besar alur kerja aplikasi web secara menyeluruh.

## Penutup

Belajar HTML, CSS, JavaScript, dan Express.js adalah proses bertahap yang saling melengkapi. Setiap teknologi memiliki peran penting dalam membangun aplikasi web yang utuh.

Melalui artikel ini, saya ingin mendokumentasikan perjalanan belajar saya sekaligus menunjukkan komitmen untuk terus berkembang sebagai web developer.
